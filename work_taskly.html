    SMART Goals

    Based on smart principals I came up with goals, that can define the initial product or MVP

    Goal No 1

    Need customers to signup for a 30 day free trail so that customers can test out the application.

    Goal No 2

    We would like to make the project and calendar section more integrated and possibly located in the same place so that customers can check the projects on the calender.

    Goal No 3

    We'd like to visualize a project in progress and who is working on it, so that its easy to track the project.
